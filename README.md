# Termgraph

A command-line tool that draws basic graphs in the terminal, written in Python.

Forked this from [mkaz/termgraph] but:
- reduced it down to a command line tool,
- cleaned up the code,
- extended colors,
- simplified build with uv,
- also tweaked it to fit my needs.
